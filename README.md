Система школьного питания — это веб-приложение для автоматизации процессов организации питания в школьной столовой. Система предоставляет различные интерфейсы для трех ролей пользователей: учеников, поваров и администраторов.

Установка:

Клонирование репозитория bash git clone <URL-репозитория> cd school_food_system
Создание виртуального окружения bash python -m venv venv venv\Scripts\activate
Установка зависимостей bash pip install -r requirements.txt
Запуск приложения bash python app.py Доступ к приложению После запуска приложение будет доступно по адресу http://localhost:8080
